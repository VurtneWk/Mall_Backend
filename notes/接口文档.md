### 公共模型

#### 01、分页请求数据

```java
{
   page: 1,//当前页码
   limit: 10,//每页记录数
   sidx: 'id',//排序字段

   key: '华为'//检索关键字
```

```java
{
	"msg": "success",
	"code": 0,
	"page": {

		"totalPage": 0,  //总页码
		"currPage": 1, //当前页码
		"list": [{  //当前页所有数据
			"brandId": 1,
			"name": "aaa",
			"logo": "abc",
			"descript": "华为",
			"showStatus": 1,
			"firstLetter": null,
			"sort": null
		}]
	}
}
```

#### 02、修改分类父子关系以及顺序

**请求方式：**

```java
POST
```

**请求地址：**

